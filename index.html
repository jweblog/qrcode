Welcome to WP-QRCODE-PLUGIN.

WP-QRCODE-PLUGIN is QRCode Wordpress plugin to automaticly create a QR code on any site of your Wordpress installation

How to use it

Use the shortcode [qrcode] within your site to automaticly generate a qrcode including the URL of the current site

Use the fullowing short code to generate a indivdual qr code with any text content:

[qrcode content="CONTENT" size="120" alt="ALT_TEXT" class="CLASS_NAME"]

Possible parameters are: alt, size, class, credit, shadow

If you don`t give any parameter like 'alt' or 'size', the standard parameters are: alt = "Scan the QR Code" ,size = 120 ,class="", credit = true, shadow = true.

The credit option gives a really small but nice looking image link on bottom of the qrcode. If you don´t like it or don´t want to give me any credits you can deactivate it with 'credit = false'.

Look at http://qrcode.jar.io for further information and for a easy useable qr code online generator.
